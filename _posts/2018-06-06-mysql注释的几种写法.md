 MySQL的注释风格总的来说有三种,它们分别是 

 

1、单行注释可以用"#",#的转化url编码是%23

```mysql
SELECT * FROM users WHERE id = '1' # 这是使用#进行注释
```

2.使用--（空格） 进行注释  在注入中可以使用--+

```mysql
SELSECT * FROM users WHERE id ='1' -- 记住--后面是有一个空格的`
```

3、多行注释可以用/**/

```mysql
SELECT * FROM users WHERE id = '1' /* 中间的就是注释的内容*/
```
